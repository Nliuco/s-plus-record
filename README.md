英雄联盟本赛季 S+ 英雄追踪工具。它会读取你本机英雄联盟客户端的熟练度数据（通过 LCU），把“本赛季最高等级/是否达到 S+”整理成一个可搜索、可筛选、可排序的英雄列表。

## 主要功能

- 从 LCU 获取本机英雄本赛季最高熟练度等级（含 `S+`/`A`/`B`/`C`/`D`）
- 视图切换：全部英雄 / 仅 S+ / 非 S+
- 搜索：官方中文名 / 外号（内置）/ 自定义别名（本机保存）
- 等级筛选：在搜索框里直接输入 `S+`、`S-`、`A+`、`B+`、`C`、`D-` 等，按本赛季等级精确匹配
- 列表排序：本赛季评分从高到低 / 从低到高
- 英雄管理：
  - 管理别名（本机保存）
  - 星标置顶（本机保存）
- 主题切换与窗口布局预设

## 使用前提

- 需要本机安装并登录英雄联盟客户端
- 建议在启动该工具前先确保客户端已运行并能被 LCU 正常读取

## 快速开始（开发）

在项目根目录执行：

```bash
npm install
npm run dev
```

## 构建与打包（Windows）

```bash
npm run dist:win
```

产物在 `release/` 目录下：

- `S+ Record-<version>-x64.exe`（NSIS 安装版）
- `S+ Record-<version>-portable-x64.exe`（便携版）

## LCU 凭证说明（隐私）

- 工具会使用 LCU 的连接信息（端口与 token）来读取本机数据
- 若你在“记住到本机”相关选项开启时，凭证会写入本机用于后续读取（具体存储方式在应用内置说明中会给出）
